# JATIC Reference Implementation

The JATIC Reference Implementation package, `jatic_ri` is set of tools to make implementing the suite of JATIC products easier.

## Description

**What is the Reference Implemenation (RI)?**
The Reference Implementation (RI) is series of UI applications which makes testing and evaluation of models and datasets straightforward for individuals who are not familar with the JATIC tools and do not have a coding background. To learn more please visit our [published documents](https://jatic.pages.jatic.net/reference-implementation/reference-implementation/).
